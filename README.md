## ğŸš€ Installation Steps

### Requirements
- XAMPP Server

1. **Clone the Repository**

    ```bash
    git clone https://github.com/NidhiDN/pet-shop.git
    ```

2. **Navigate to Project Directory**

    ```bash
    cd pet-shop
    ```

3. **Place the Project in XAMPP's `htdocs` Folder**

    Move the `pet-shop` folder to your `XAMPP/htdocs` directory.

4. **Start XAMPP Server**

    Open the XAMPP control panel and start the Apache and MySQL services.

5. **Access the Dashboard**

    Open a web browser and navigate to `http://localhost/pet-shop`.
